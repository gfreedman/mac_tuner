MacTuner

Scans your Mac for security and health issues. Explains what it finds. Fixes what it can.

$ brew install gfreedman/mactuner/mactuner

Install

Homebrew easiest
brew tap gfreedman/mactuner
brew install mactuner
pipx
pipx install mactuner

pipx keeps command-line tools isolated from your Python setup. Good choice if you use Python for other things.

pip
pip3 install --user mactuner

Needs macOS 13 Ventura or later and Python 3.10 or later. Works on Intel and Apple Silicon.

First run

Open Terminal, type mactuner, and press Enter.

MacTuner runs 69 checks, prints each result with a short explanation, and gives your Mac a score out of 100. It takes about 30 to 45 seconds.

MacTuner is read-only by default. Nothing on your Mac changes until you run mactuner --fix.

Commands

Command What it does
mactuner Full scan
mactuner -y Full scan, skip the start prompt
mactuner --issues-only Show only warnings and criticals
mactuner --explain Extra context per finding
mactuner --only security Scan one category
mactuner --only security,disk Scan multiple categories
mactuner --skip dev_env Skip a category
mactuner --fix Step through fixable issues after the scan
mactuner --fix --auto Apply safe fixes without asking each time
mactuner --quiet Print only the score
mactuner --json Output as JSON for scripts
mactuner --check-shell-secrets Scan shell configs for hardcoded credentials

Category names for --only and --skip

system  security  privacy  homebrew  disk  hardware  memory  network  dev_env  apps

Fix mode

Run mactuner --fix after a scan to go through fixable issues one at a time.

For each issue, MacTuner shows what it found, why it matters, and exactly what the fix does. You approve or skip it before moving on.

Safety

  • Read-only by default
  • Runs entirely offline
  • One failing check never stops the scan
  • Every command shown before it runs
  • Irreversible fixes always labelled
  • No private APIs, no TCC.db access
  • Detects managed Macs and flags MDM settings
  • Works on Intel and Apple Silicon
