MacTuner

Scans your Mac for security and health issues. Explains what it finds. Fixes what it can.

$ brew install gfreedman/mactuner/mactuner

Install

Homebrew easiest
brew tap gfreedman/mactuner
brew install mactuner
pipx
pipx install mactuner

Good if you already use Python tools. Keeps mactuner isolated.

pip
pip3 install --user mactuner

Works everywhere Python 3.10+ is installed.

Needs macOS 13 Ventura or later. Python 3.10 or later. Intel and Apple Silicon.

First run

Open Terminal, type mactuner, and press Enter. MacTuner runs 69 checks, explains each one, and gives your Mac a score out of 100. Takes about 30 to 45 seconds.

MacTuner is read-only by default. Nothing changes until you run mactuner --fix.

Commands

Command What it does
mactunerFull scan
mactuner -yFull scan, no start prompt
mactuner --issues-onlyShow only warnings and criticals
mactuner --explainExtra context per finding
mactuner --only securityScan one category
mactuner --only security,diskScan multiple categories
mactuner --skip dev_envSkip a category
mactuner --fixStep through fixable issues after the scan
mactuner --fix --autoApply safe fixes without prompting each time
mactuner --quietPrint only the score
mactuner --jsonOutput as JSON for scripts
mactuner --check-shell-secretsScan shell configs for hardcoded credentials

Category names

system  security  privacy  homebrew  disk  hardware  memory  network  dev_env  apps

Use these with --only or --skip. Combine with commas: --only security,disk

Fix mode

Run mactuner --fix after a scan to go through fixable issues one at a time. For each one, MacTuner shows what it found, why it matters, and exactly what the fix does. You approve or skip before anything runs.

Safety

  • Read-only by default
  • Runs entirely offline
  • One failing check never stops the scan
  • Every command shown before it runs
  • Irreversible fixes always labelled
  • No private APIs, no TCC.db access
  • Detects managed Macs, flags MDM settings
  • Works on Intel and Apple Silicon
